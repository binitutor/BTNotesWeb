# BTNotesWeb - Todo List App

Overview
BTNotes is a lightweight todo and notes frontend with authentication views, task management, and team project tracking. The UI is responsive and styled with Bootstrap 5.3, featuring a light theme and priority labels.

Features
- Authentication pages: dedicated Login and Signup screens with form validation and error messaging.
- Protected routing: unauthenticated users are redirected to Login before accessing the dashboard.
- Task management: add tasks, plan notes, and password-protected notes with priority, due date, and description.
- Task list and cards: view tasks as cards and in a DataTable with sorting and quick actions.
- Status toggles: mark tasks as pending or complete directly from cards or table.
- Delete actions: remove tasks with confirmation prompts.
- Progress tracking: Chart.js doughnut chart summarizes completed vs pending tasks.
- Team project tracking: sample project table with progress indicators.
- Team collaboration list: quick view of team members and focus areas.
- Notifications: SweetAlert2 feedback for add, delete, and auth actions.
- Responsive design: mobile-friendly layout and components.
